یک آزمایش کوچک: دو نسخه از یک لوگوموشن را به ده نفر نشان دادیم. گرافیک، رنگ و فونت هر دو یکی بود؛ فقط منحنی حرکت فرق داشت. نه نفر نسخهٔ دوم را «حرفه‌ای‌تر» دانستند، اما هیچ‌کدام نتوانستند بگویند چرا.

چرایش ساده است: چشم ما تایمینگ بد را فوراً می‌فهمد، حتی اگر زبانمان برای توصیفش کلمه نداشته باشد.

## حرکت طبیعی یعنی چه؟

در دنیای واقعی هیچ چیزی با سرعت ثابت حرکت نمی‌کند. ماشین آرام راه می‌افتد، سرعت می‌گیرد، و نرم ترمز می‌کند. توپ می‌افتد، می‌پرد، و هر پرش کوتاه‌تر از قبلی است. مغز ما یک عمر با این قوانین زندگی کرده.

وقتی یک عنصر گرافیکی با سرعت ثابت (اصطلاحاً linear) حرکت می‌کند، مغز می‌گوید: «این طبیعی نیست، ماشینی است.» و همین حس ناخودآگاه، کل کار را ارزان جلوه می‌دهد.

## سه قانونی که بیشترین تفاوت را می‌سازند

### ۱. ورود تند، توقف نرم

برای عناصری که وارد قاب می‌شوند، تقریباً همیشه ease-out جواب می‌دهد: سریع شروع کن، نرم بایست. مخاطب باید حرکت را زود ببیند و توقف را حس نکند.

### ۲. هیچ‌وقت همه‌چیز را با هم تکان نده

اگر پنج عنصر دقیقاً با هم وارد شوند، چشم نمی‌داند کجا را نگاه کند. فاصلهٔ ۵۰ تا ۱۰۰ میلی‌ثانیه‌ای بین عناصر (stagger) همان چیزی است که به کار ریتم و نفس می‌دهد.

### ۳. سکوت هم بخشی از ریتم است

موشن خوب مثل موسیقی خوب، مکث دارد. بین دو صحنهٔ پرتحرک، یک لحظه سکون بگذارید تا پیام بنشیند. انیمیشنی که یک ثانیه آرام نمی‌گیرد، مخاطب را خسته می‌کند.

## یک مثال کدِ واقعی

همین سایتی که الان در آن هستید از همین اصول استفاده می‌کند. این الگوی GSAP برای ورود کلمه‌به‌کلمهٔ تیتر است:

```js
gsap.fromTo(
  words,
  { yPercent: 110, opacity: 0 },
  {
    yPercent: 0,
    opacity: 1,
    duration: 0.7,
    stagger: 0.06, // فاصلهٔ کوتاه بین کلمه‌ها = ریتم
    ease: "power3.out", // ورود تند، توقف نرم
  }
)
```

دو عدد مهم اینجا `stagger` و `ease` هستند. همین دو را تغییر بدهید تا ببینید حس کار چقدر عوض می‌شود — بدون اینکه یک پیکسل از گرافیک تغییر کند.

## چک‌لیست قبل از تحویل هر موشن

- [ ] هیچ حرکتی linear نیست، مگر عمداً (مثل چرخش دائمی یا مارکی)
- [ ] عناصر هم‌خانواده با stagger وارد می‌شوند، نه یک‌جا
- [ ] بین بخش‌های پرتحرک، لحظهٔ سکون وجود دارد
- [ ] مدت انیمیشن‌های رابط کاربری زیر یک ثانیه است
- [ ] حرکت، معنا دارد؛ چشم را به سمت پیام اصلی می‌برد، نه فقط «قشنگ است»

تایمینگ مهارتی یادگرفتنی است، اما میانبر ندارد: باید زیاد ببینید، زیاد بسازید و هر بار از خودتان بپرسید: «اگر این شیء واقعی بود، چطور حرکت می‌کرد؟»
